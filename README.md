# SteamWave
This is a implementation of SteamVave streaming service built with React.

## Description
This is a demo version of SteamVave which manages content streaming, searching and creation of playlist. 

## Installation
To install navigate to react-demo then install using npm install

```bash
cd react-demo
npm install
```
## Run the code
To run the demo code use

```bash
npm run dev
```
